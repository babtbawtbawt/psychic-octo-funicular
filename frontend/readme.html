Khanworld readme

Khanworld is a BonziWORLD server created by donutscout. It is a free-standing server that is not part of the BonziWORLD server. It is a simple server that is meant to be a less clusterfucked version of the BonziWORLD server meant for development. If you want it to be free-standing, go ahead but I recommend making some changes to personalize it.

This is mostly intended as a deprecation of the ol' CyganWORLD into a new version that WILL be updated. Reminding you that cyganworld is very, very insecure and should not be used in production. You will need Express, CORS, and socket.io (obviously) This is very unfinished right now.
